The core of this application is a custom JavaScript function that performs a **Data Transformation** task:

1. It captures user-input strings from four distinct DOM elements.
2. It validates the presence of data to ensure prompt quality.
3. It performs a structured concatenation, injecting the inputs into a proven prompt-engineering template.
4. It outputs the final string back to the DOM for user retrieval.
